# chattr

## fullstack online chat wep application 

technologies used in project:

	- frontend:
		- angular
		- angularSSR
		- NgRx
		- angular cdk
		- tailwindCSS
		- spartanUI
	- backend:
		- nestJS
		- postgreSQL
		- prismaORM
		- webSockets (using socket.io)
		- JWT

### for now it only has authentication and authorization logic implemented, but soon will be added more features :)
